# ScreenPager component for react-native

## Installation
```npm install react-native-screenpager```
and install the peerDependencies listed in package.json

## Usage

```javascript
import ScreenPager from 'react-native-screenpager';

<ScreenPager>
  <YourViewA />
  <YourViewB />
</ScreenPager>
```
